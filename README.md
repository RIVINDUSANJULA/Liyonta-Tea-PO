## 📖 Overview

The **Liyonta Tea PO System** streamlines the B2B order process by combining a modern, interactive frontend with a robust REST API and a secure database. Designed with a strict separation of concerns, the application ensures high performance, real-time document rendering, and seamless inventory management without the need for page reloads.

## ✨ Key Features

* 📄 **Real-Time Document Generation:** Features a live, side-by-side PDF preview and high-fidelity rendering engine using `@react-pdf/renderer`.
* 🎛️ **Single Page Admin Dashboard:** A tabbed interface allows administrators to manage client lists and tea grades dynamically.
* 🧮 **Dynamic Order Processing:** Instantly calculates subtotals, manages itemized rows, and pulls product data directly from a live SQL database.
* 🏗️ **Decoupled Architecture:** A standalone Node.js/Express backend API powers the Next.js frontend, ensuring maximum scalability and security.
* 🎨 **Premium UI/UX:** Crafted with Tailwind CSS, featuring subtle animations, glass-morphism effects, and a highly polished brand aesthetic suitable for an enterprise environment.

## 🛠️ Technical Architecture

### Frontend (Client-Side)
* **Framework:** Next.js (App Router)
* **Library:** React 18
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **PDF Engine:** React-PDF

### Backend (Server-Side & API)
* **Runtime:** Node.js
* **Framework:** Express.js
* **Middleware:** CORS, Express JSON
* **Database Driver:** `mysql2` (Connection Pooling)

### Database
* **Engine:** MySQL 8.0+
* **Structure:** Relational Reference Data (Customers, Tea Grades)
